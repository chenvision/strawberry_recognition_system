# 草莓位姿估计系统：数据流水线与实验设置说明

## 1. 模型张量维度演变推导 (Tensor Shape Propagation)

本系统采用单阶段网格化回归架构，输入图像分辨率为 $800 \times 600$。下表详细推导了数据张量在网络各层间的维度变化过程（假设 Batch Size 为 $B$）：

| 阶段 / 网络层 | 操作类型 (Stride) | 输出张量维度 (C, H, W) | 空间缩放比例 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **输入层 (Input)** | 图像读取与预处理 | $[B, 3, 600, 800]$ | $1 \times$ | 原始 RGB 图像张量 |
| **Backbone Block 1** | Conv + Pool (2) | $[B, 32, 300, 400]$ | $1/2$ | 初步特征提取 |
| **Backbone Block 2** | Conv + Pool (2) | $[B, 64, 150, 200]$ | $1/4$ | - |
| **Backbone Block 3** | Conv + Pool (2) | $[B, 128, 75, 100]$ | $1/8$ | - |
| **Backbone Block 4** | Conv + Pool (2) | $[B, 256, 37, 50]$ | $1/16$ | 空间维度进一步压缩 |
| **Backbone Block 5** | Conv + Pool (2) | $[B, 512, 18, 25]$ | $1/32$ | 达到预定网格分辨率 |
| **Backbone Block 6** | Conv (1) | $[B, 1024, 18, 25]$ | $1/32$ | 深度特征融合 |
| **Transformer Neck** | 1x1 Conv + Attn | $[B, 256, 18, 25]$ | $1/32$ | 通道降维并执行全局增强 |
| **Prediction Head** | 1x1 Conv (1) | $[B, 22, 18, 25]$ | $1/32$ | 最终 22 维回归参数映射 |

**维度变换关键点：**
1.  **下采样倍率**：网络总计执行了 5 次步长为 2 的最大池化操作，总下采样倍率为 $32$。
2.  **网格对齐**：输入宽 $800/32 = 25$，高 $600/32 \approx 18$（执行了向上/向下取整对齐），生成了 $18 \times 25$ 的特征网格，每个网格单元负责覆盖原始图像中 $32 \times 32$ 像素的区域。

## 2. 实验设置与预处理学术描述 (用于论文实验章节)

**章节标题：数据预处理与模型训练设置**

> **学术表述：**
> 本研究在 Straw6D 数据集上验证了所提算法的有效性。为了增强模型对复杂果园环境（如光照剧烈变化、枝叶遮挡等）的鲁棒性，系统在训练阶段实施了多重数据增强策略。
>
> **2.1 数据预处理与增强 (Data Augmentation)**
> 所有原始图像均被重采样至 $800 \times 600$ 像素的统一分辨率。在数据馈入网络前，应用了色彩抖动（Color Jittering）策略，随机调整图像的亮度（Brightness, $\pm 0.2$）、对比度（Contrast, $\pm 0.2$）、饱和度（Saturation, $\pm 0.2$）及色调（Hue, $\pm 0.1$），以模拟不同时间段及天气条件下的自然光照环境。图像像素值通过归一化操作被线性映射至 $[0, 1]$ 空间，并转换为符合 PyTorch 规范的浮点型张量。
>
> **2.2 标签网格化编码 (Grid-based Label Encoding)**
> 针对三维位姿估计任务，本实验采用了一种空间网格化编码方案。系统将图像平面划分为 $18 \times 25$ 的特征网格，若草莓目标的投影中心落入某一特定网格，则该网格负责回归目标的 22 维关键参数（包括 3D 边界框关键点偏移量及物理尺寸）。
>
> **2.3 训练配置 (Hyper-parameters)**
> 本算法基于 PyTorch 框架实现，硬件环境采用 NVIDIA Tesla 系列 GPU。训练过程中使用 Adam 优化器，初始学习率设定为 $10^{-3}$。批处理大小（Batch Size）配置为 16，总迭代周期（Epoch）设置为 300。损失函数采用加权多任务损失，包括 Smooth-L1 回归损失与二元交叉熵（BCE）置信度损失，权重系数分别设定为 $\lambda_{coord}=1.0, \lambda_{dim}=0.5, \lambda_{conf}=1.0$。
