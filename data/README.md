# Straw6D 草莓三维姿态与尺寸估计数据集说明

## 1. 项目背景
本数据集（Straw6D）是针对单目视觉环境下草莓果实的三维位姿（Position & Orientation）及尺寸（Size）估计任务而设计的。
它提供了丰富的多模态信息，支持从传统的 2D 目标检测到高级的 6D 姿态估计（6-DoF Pose Estimation）研究。

## 2. 数据集目录结构

```text
data/
├── Straw6D_Raw/                # 原始数据集核心目录
│   ├── images/                 # [RGB 图像] 包含草莓场景的彩色图 (.png)
│   ├── depth_images/           # [深度图] 与 RGB 对应的深度信息图，用于 3D 空间反投影
│   ├── labels_maps/            # [标签图] 实例分割或分类的掩膜图
│   ├── colored_maps/           # [可视化标签] 经过伪彩色处理的标签图，便于人工核对
│   ├── boxes/                  # [3D 标注数据] 核心 CSV 文件，记录了每个果实的 3D 参数
│   └── view_dataset.py         # 数据集可视化工具脚本
└── Straw6D_Upload/             # 云端/Kaggle 上传预备目录
```

## 3. 核心标注说明 (boxes/ 目录)
每个 CSV 文件（如 `0000.csv`）对应一张图片中的所有草莓目标，字段含义如下：

| 字段 | 含义 | 说明 |
| :--- | :--- | :--- |
| **label** | 类别/状态 | 草莓的不同成熟度或分类标签 (e.g., 250, 125) |
| **x, y, z** | 三维坐标 | 目标中心点在相机坐标系下的空间位置 (单位：米) |
| **w, h, l** | 三维尺寸 | 果实的 宽 (Width), 高 (Height), 长 (Length) (单位：米) |
| **roll, pitch, yaw** | 欧拉角 | 目标在三维空间中的旋转姿态（翻滚、俯仰、偏航角） |

## 4. 数据特征
- **高维度信息**：不同于常规 2D 数据集，Straw6D 提供了完整的空间几何信息（6-DoF）。
- **同步 RGB-D**：支持将 2D 图像特征与 3D 深度特征结合，进行多模态融合学习。
- **实际应用导向**：专门为农业机器人采摘草莓时的精准定位和尺寸分级场景设计。

## 5. 快速预览
你可以运行以下脚本来可视化数据集的标注效果：
```powershell
python data/Straw6D_Raw/view_dataset.py
```
