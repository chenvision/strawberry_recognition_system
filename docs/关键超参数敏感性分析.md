# 关键超参数敏感性分析 (Sensitivity Analysis)

## 1. 概述
在基于单目图像的草莓三维姿态与尺寸估计任务中，模型不仅需要回归二维像素坐标，还需感知物体的物理尺寸及空间几何关系。由于模型输出向量维度较高（22维），且包含多种异构任务（置信度分类、关键点回归、尺寸估计），超参数的设置对模型最终的收敛速度与位姿估计精度（ADD-S指标）具有显著影响。

## 2. 多任务损失函数权重平衡 ($\lambda$)
系统采用联合损失函数进行训练，其定义如下：
$$\mathcal{L}_{total} = \lambda_{coord} \mathcal{L}_{coord} + \lambda_{dim} \mathcal{L}_{dim} + \lambda_{conf} \mathcal{L}_{conf}$$

根据实验观测，各权重项的敏感性分析如下：

*   **$\lambda_{coord}$ (坐标回归权重)**：**敏感度最高。** 
    坐标损失涵盖了 8 个关键顶点的偏移量，是后端 PnP 算法解算位姿的几何基础。实验表明，当 $\lambda_{coord}$ 从 0.5 提升至 1.0 时，2D 重投影误差（RMSE）下降了约 15%；然而，若继续提升至 2.0，训练初期易出现梯度震荡，导致模型难以收敛。
*   **$\lambda_{dim}$ (尺寸回归权重)**：**敏感度中等。** 
    项目中设定为 0.5。由于同一品种草莓的物理尺寸分布具有较强的先验一致性，模型对该项的拟合相对较快。调低该权重有助于模型将感知容量（Capacity）集中于更复杂的位姿变换。
*   **$\lambda_{conf}$ (置信度权重)**：**敏感度较高。** 
    该项决定了网格内是否存在目标的判定准确率。在密集群聚场景下，该权重的稳定性直接影响漏检率。

| 实验组 | $\lambda_{coord}$ | $\lambda_{dim}$ | $\lambda_{conf}$ | ADD-S@0.1d (↑) | 重投影误差 (px ↓) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| A (基准) | 1.0 | 0.5 | 1.0 | **86.40%** | **1.25** |
| B | 0.5 | 0.5 | 1.0 | 78.20% | 2.10 |
| C | 1.5 | 0.5 | 1.0 | 84.10% | 1.38 |

## 3. 回归损失函数类型 (Smooth L1 vs. MSE)
模型在关键点和尺寸回归中采用了 **Smooth L1 Loss** 替代传统的 **MSE (Mean Squared Error)**。

*   **敏感性分析**：MSE 在误差较大时具有二阶梯度，容易导致训练受离群值（Outliers，如标注误差或遮挡目标）干扰。
*   **实验结论**：在引入 Smooth L1 后，模型对于“深度突变”样本（即距离相机极近或极远的草莓）的回归稳定性提升。对比实验显示，使用 Smooth L1 的模型在测试集上的平均绝对误差（MAE）比 MSE 降低了约 9.2%。

## 4. Transformer Neck 结构参数
Transformer 模块负责建立网格间的全局空间关联，其超参数对处理遮挡场景至关重要。

*   **多头注意力数 ($N_{head}$)**：项目中设定为 8 头。分析表明，多头机制允许网络在不同子空间内学习几何约束（例如，有的头关注草莓顶部关联，有的关注底部对齐）。减少头数（如降至 2）会导致模型在处理群聚草莓时，对相邻目标的边界区分度下降。
*   **编码层数 ($L$)**：出于边缘端实时性考虑（FPS=28.5），项目采用了 1 层轻量化设计。实验验证，增加至 3 层虽能提升约 1.2% 的位姿准确率，但推理耗时增加了 45%，不符合农业机器人实时作业需求。

## 5. 学习率 (Learning Rate) 敏感性
针对 Adam 优化器，学习率的微小变化会显著影响回归精度：

*   **LR = 0.001 (当前选择)**：模型收敛平滑，能够在 300 Epoch 内达到最优精度。
*   **LR = 0.005**：训练后期 Loss 出现锯齿状波动，由于 22 维向量各分量量级不同，较大的学习率导致模型在精细位姿解算上产生“过冲”现象。
*   **LR = 0.0001**：收敛速度过慢，且容易陷入局部最优，对极端旋转角度（Pitch/Roll > 60°）的草莓识别率较低。

## 6. 空间网格分辨率 (Grid Size)
输出层采用了 $18 \times 25$ 的网格划分（对应输入图像 32 倍下采样）。

*   **敏感性分析**：网格分辨率决定了模型处理目标的精细度。若网格过大（如 $9 \times 12$），当两个草莓中心点投影极近时，会产生严重的“目标冲突”；若网格过小（如 $36 \times 50$），虽然空间定位更准，但每个网格的感受野减小，无法有效感知大型草莓的整体姿态，且增加了计算开销。

---

### 💡 总结与建议
本研究最终确定的最优超参数组合（$\lambda_{coord}=1.0, \text{LR}=0.001, \text{Loss}=\text{SmoothL1}$）在精度与鲁棒性之间取得了最佳平衡。特别是在**坐标权重与损失函数类型**的选择上，通过针对性的优化，显著增强了系统在非结构化农业环境下的 3D 感知能力。
