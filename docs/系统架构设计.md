# 草莓单目三维姿态与尺寸估计系统：总体方案设计

## 1. 系统整体数据流向 (Pipeline)

本系统采用“单阶段网格预测 + 几何位姿解算”的总体思路。完整的数据处理流程如下：

1.  **图像预处理层**：将输入的原始单目 RGB 图像统一缩放至 $800 \times 600$ 像素，进行归一化处理并转换为 4 维张量（Batch, Channel, Height, Width）送入深度神经网络。
2.  **特征提取与增强层 (Backbone & Neck)**：
    *   **主干网络 (Darknet-19)**：通过多层卷积与池化操作提取草莓目标的局部颜色、纹理及高级语义特征。
    *   **增强网络 (Transformer Neck)**：在特征图末端引入轻量化 Transformer 编码器，利用自注意力机制（Self-Attention）捕捉全局上下文信息，强化网格单元间的位置关联。
3.  **多任务网格预测层 (Prediction Head)**：
    *   网络将特征图划分为 $18 \times 25$ 的空间网格（Grid）。
    *   每个网格单元并行回归一个 **22 维预测向量**：包含中心点坐标 $(u, v)$、3D 边界框 8 个顶点的投影偏移量（16 维）、草莓三轴物理尺寸 $(l, w, h)$ 以及目标置信度。
4.  **后处理与几何解算层**：
    *   **目标筛选**：利用置信度阈值初筛，并采用基于欧氏距离的非极大值抑制（NMS）算法剔除重叠检测。
    *   **位姿求解 (PnP)**：将预测的顶点 2D 坐标与预设 3D 参考模型匹配，结合相机内参矩阵，利用 **EPnP 算法** 求解草莓相对于相机的旋转向量（$R$）和平移向量（$T$）。
5.  **结果输出层**：最终输出草莓在相机坐标系下的 3D 坐标（$x, y, z$）、旋转参数及物理尺寸估算值。

## 2. 总体方案学术描述 (用于论文章节)

**章节标题：基于单阶段网格回归的草莓三维位姿与尺寸估计方案设计**

> **学术表述：**
> 本研究提出了一种端到端的单目图像草莓三维信息感知框架。方案核心在于将复杂的位姿估计任务转化为空间网格上的多任务并行回归问题。系统构建了一个以改进型 Darknet-19 为主干特征提取器、并引入 Transformer 编码器作为特征增强模块的深度神经网络架构。该架构能够直接从原始像素空间提取高维语义特征，并将其映射至预定义的网格空间。
>
> 在模型输出端，系统通过回归头并行解析目标中心点、三维边界框投影顶点及物体的三轴物理尺寸。针对单目视觉中深度信息缺失的挑战，本方案引入了投影几何约束机制，将深度神经网络输出的二维关键点与三维物体空间坐标通过 PnP (Perspective-n-Point) 算法进行关联，从而实现物体 6D 位姿（即三维旋转与平移）的精确解析。该方案有效避免了传统双阶段算法在区域提议阶段的计算冗余，在确保草莓位姿估计精度的前提下，提升了系统在实时场景中的处理效能。

## 3. 系统架构图逻辑结构建议

在绘制系统架构图时，建议采用以下模块化逻辑：

### 架构模块流向图
**[图像输入层]** $\rightarrow$ **[特征提取层 (Backbone)]** $\rightarrow$ **[特征增强层 (Neck)]** $\rightarrow$ **[多任务预测层 (Head)]** $\rightarrow$ **[几何推理层]** $\rightarrow$ **[结果输出层]**

### 模块功能定义

| 模块层级 | 核心组件 | 功能说明 |
| :--- | :--- | :--- |
| **图像预处理层** | Resize, Normalization | 统一输入规范，适配网络输入维度。 |
| **特征提取层** | Darknet-19 卷积堆叠 | 提取草莓目标的底层图像特征与高层语义特征。 |
| **特征增强层** | Transformer Encoder | 利用全局注意力机制提升目标在复杂背景下的辨识度。 |
| **网格预测层** | 1x1 卷积回归头 | 在 $18 \times 25$ 网格内并行预测位姿与尺寸相关参数。 |
| **几何推理层** | EPnP 算法, 相机内参 | 利用几何投影模型将 2D 关键点转化为 3D 空间位姿（R, T）。 |
| **后处理层** | 置信度过滤, 距离 NMS | 消除冗余检测，优化输出结果的鲁棒性。 |
