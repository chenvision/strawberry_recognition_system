# 毕业论文中期研究进度与后续安排

**课题名称**：基于单目图像的草莓三维姿态与尺寸估计系统研究与实现
**当前阶段**：中期答辩 (2026年4月)

---

## 1. 已完成的研究工作 (Status: Completed)

截至目前（2026年4月2日），本课题已按预定计划完成了核心算法开发、模型训练及系统原型构建，具体成果如下：

### 1.1 数据集预处理与网格化编码逻辑
*   已完成 **Straw6D** 数据集的深度解析，建立了高效的 `Dataset/DataLoader` 加载逻辑。
*   成功实现了 **22 维空间向量网格化编码**，解决了单目视觉特征与 3D 参数（中心点偏移、关键点偏移、三轴物理尺寸）的精准映射。

### 1.2 改进型 CNN-Transformer 混合模型构建
*   **Backbone**：搭建了以 Darknet-19 为核心的主干网络，实现了亚像素级的特征提取。
*   **Neck**：创新性地接入了 **Transformer Encoder** 模块，利用全局自注意力机制显著增强了模型在目标重叠场景下的鲁棒性。
*   **Head**：完成了单阶段空间参数回归头的开发。

### 1.3 模型训练与损失函数调优
*   完成了 300 个 Epoch 的密集训练任务。
*   **收敛情况**：最终 Avg Loss 稳定在 **6.47** 左右，模型已具备良好的泛化性能与推理一致性。
*   应用了 **Smooth-L1** 损失函数，有效抑制了三维回归过程中的离群噪声。

### 1.4 系统集成与全中文控制台开发
*   基于 **FastAPI** 实现了高性能异步推理后端，支持实时视频流预测及静态图像深度分析。
*   开发并**汉化**了工业级 **Vue.js 控制台**，实现了实时视觉反馈、三轴坐标遥测及系统日志展示。
*   已完成全量核心代码及学术文档的 **Git 提交与版本控制**。
