# 《基于统计机器学习的草莓 3D 形态与姿态状态分类 Baseline》

## 1. 项目简介
本项目是《统计机器学习》课程的结课作业。研究目标是通过分析草莓的 3D 几何特征（宽、高、长、体积、长宽比）以及空间姿态（Roll, Pitch, Yaw），实现对草莓果实状态/类位的精准分类。

模型采用**随机森林 (Random Forest)** 算法，结合 **PCA (主成分分析)** 进行特征降维可视化。项目直接读取 Straw6D 数据集中的 2000+ 个真实 6-DoF 标注文件，具有极高的工程实战价值。

## 2. 核心技术点
- **大规模数据处理**：利用 `glob` 与 `tqdm` 稳健地处理 2000+ 个 CSV 文件的合并与清洗。
- **特征工程**：基于原始位姿数据计算果实三维体积与多维度长宽比，提升模型表征能力。
- **统计机器学习管线**：数据标准化 -> PCA 降维分析 -> 随机森林多分类。

## 3. 运行指南 (一键运行)
代码支持跨目录自动寻找绝对路径下的数据集，助教只需执行以下指令：

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行核心分析脚本
python main.py
```

## 4. 实验产出
运行结束后，可在 `results/` 目录下查看：
- `pca_scatter.png`: 展示 2000+ 样本在特征空间中的分布。
- `confusion_matrix.png`: 定量分析模型在不同类别上的准确性。
